stepper motor
->
unipolar/bipolar stepper motor



BLDC, servo 모터와 구조가 비슷하고,
BLDC 모터와 원리가 비슷하다.

제어 방식은 자체 feedback이 없는 open loop이다.

servo보다 토크가 작다.





데이터시트에서 5가지를 확실히 기억하자.



1.
스텝 방식과 앵글

stem angle: 1.8degree (+- 5%@ full step. no load)

full step.
->
스텝모터 구동 방식으로 상여자 방식이 있다. (여자: 전자석에 전류가 흘러 자기화(자석)되는 것.)



1상/2상/1-2상 여자방식.

1상 여자방식 (one phase full-step) : bldc motor처럼, 둘러싼 권선이 순차적으로 자기화된다. (차례로 상에 전기를 준다.)
->
모터의 과열이 적다.
각도가 정밀하다.
진동이 심하다.

2상 여자방식 (two phase full-step) : 동시에 2개의 상에 전기를 준다.
->
전류 소모가 2배이고, 힘도 2배이다. (아마도)
진동이 적다.
주로 사용되는 스텝모터이다.

1-2상 여자방식 (one-two phase half-step) : 1상과 2상 여자방식을 교대로 사용한다.
->
각 4개의 상이 켜지는 순서 : 12 -> 2 -> 23 -> 3 -> 34 -> 4 -> 41 -> 1 -> ...
각도를 반으로 쪼개어 더 정밀하다.





2.
모터의 이름

NEMA 17 . 23

NEMA : 국제 표준규격을 따른다는 것.

17 : 모터 내부 권선이 아닌 제품 밑면에서 한 변의 길이이다.

23 : 모터의 높이.





3.
모터의 힘 (holding torque, 홀딩 토크)

모터의 최대 하중.

Nm (뉴턴미터) : 1m 떨어진 곳에서 들어올릴 수 있는 힘.

// 물체 고유의 질량 단위가 kg, 중력을 포함한 단위가 kgf (force) ( == N)
// n.m -> kgf.xcm로 변환 : 나누기 중력가속도 9.8, 곱하기 100/x
// e.g. 0.5n.m -> 1kgf.5cm
// 뭐든간 성능의 70~80%만 사용하는 것이 좋다. 모터 및 드라이버, 더 나아가 보드가 통구이가 될 수 있다.





4.
전류량(암페어)

전압/저항이 모두 제각각이다.
전류량은 모터드라이버와 매치할 때 꼭 체크해야 한다.




5.
권선 방식

유니폴라
->
권선에 전류를 한 쪽 방향으로만 흘리는 방식이다.
작은 토크(회전력, 힘), 낮은 탈조 위험(적은 전압/전류 환경에서 정밀 제어 가능), 고속 회전 가능.
// 저속에서 힘이 좋다.

바이폴라
->
권선에 흐르는 전류의 방향을 바꿔서 모터를 구동한다.
회로구성이 복잡하며, 유니폴라보다 전체적으로 정밀도가 높다.
// 고속에서 힘이 좋다.

4개의 전자석으로 둘러싼 4선식은 바이폴라만,
6선식은 유니폴라/바이폴라 방식 모두 있다.








6.
스테퍼 모터 드라이버
// https://www.youtube.com/watch?v=iIs82XkTvhY&t=24s

// SMPS (switching mode power supply) : 스위칭 동작에 의한 전원공급장치이다. 일반적으로 파워 서플라이라고 불리며, 넓은 범위의 입력전압으로 일정하고 안정적인 출력전압을 제공해주는 기기이다.

모터의 동작과 직결되어 모든 것을 결정하기 때문에, 모터 규격에 맞는 드라이버를 선택하는 것이 매우 중요하다.

종류가 너무 많다.

대중적으로 사용되는 모델 두 개
->
TMC2208
TMC2209
-> 각각 업그레이드되어 출시된 모델이
TMC2225
TMC2226
칩의 모양이 바뀌면서 방열 성능이 좋아졌다고 한다.



에듀이노의 스텝 모터 모델명은 28BYJ-48
호환 모터드라이버는 ULN2003
-> hybrid가 아니다.







7.
모터에 호환되는 바퀴

에듀이노 스텝 모터의 바퀴 제품은 못 찾았어.
3D printing, customizing과 같은 자체 제작 관련 내용밖에 없어.

우리가 3D프린터를 사용한다면,
사용하려는 스텝모터의 축보다 큰 구멍을 가진 바퀴를 연결할 어댑터를 여러 치수로 뽑아서 껴 봐야 해.






8.
스테퍼 모터 종류
-> http://www.npmkorea.co.kr/index.php

구조에 따라 PM, VR, HYBRID로 나뉜다.
HYBRID가 나머지 둘의 장점을 결합한, 진보된 구조이며,
관련 국제 규격으로 NEMA가 있으며, 가장 흔하고 널리 사용되는 유형이다.
특수 설계된 걸 제외하면, 대부분 NEMA 규격(이자 제품명)을 사용한다고 생각하면 된다.
